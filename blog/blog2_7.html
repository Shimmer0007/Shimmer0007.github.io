Chapter 1 启航 & Chapter 2 数据类型和操作

一、常用内置类型（Builtin Types）

  • 整数（int）：整数是没有小数部分的数字，如 1, 2, 3。

  • 浮点数（float）：浮点数是带有小数点的数字，如 2.2, 3.14。

  • 布尔值（bool）：布尔值只有两个值：True 和 False，用于逻辑判断。

  • 类型（type）：类型也是一种类型，用于表示数据的类型。可以通过 type() 函数查看。

    print(type(2))       # 输出：<class 'int'>
print(type(2.2))     # 输出：<class 'float'>
print(type(2 < 2.2)) # 输出：<class 'bool'>
print(type(type(42))) # 输出：<class 'type'>

二、常用内置常数（Builtin Constants）

  • 布尔常数

    • True：表示真
    • False：表示假
    • None：表示空值，无具体意义

  • 数学常数

    • math.pi：圆周率 π
    • math.e：自然对数的底数 e
    • math.tau：圆周率的两倍，τ
    • math.inf：无穷大
    print(math.pi)  # 输出：3.141592653589793
print(math.e)   # 输出：2.718281828459045
print(math.tau) # 输出：6.283185307179586
print(math.inf) # 输出：inf

三、常用内置运算符（Builtin Operators）

  • 算术运算符

    • `+`：加法
    • `-`：减法
    • `*`：乘法
    • `/`：除法（浮点数除法）
    • `//`：整除
    • `%`：取余
    • `**`：幂运算

  • 关系运算符

    • `==`：等于
    • `!=`：不等于
    • `<`：小于
    • `>`：大于
    • `<=`：小于等于
    • `>=`：大于等于

  • 赋值运算符

    • `=`：赋值
    • `+=`：加后赋值
    • `-=`：减后赋值
    • `*=`：乘后赋值
    • `/=`：除后赋值
    • `//=`：整除后赋值
    • `%=`：取余后赋值
    • `**=`：幂运算后赋值

  • 逻辑运算符

    • `and`：逻辑与
    • `or`：逻辑或
    • `not`：逻辑非

四、运算符优先级（Operator Precedence）

运算符的优先级决定了在表达式中运算符执行的顺序。例如，乘法运算符比加法运算符有更高的优先级，所以乘法会先执行。

result = 2 + 3 * 4 # 输出：14，因为 3 * 4 先计算

五、类型影响语义（Type Affects Semantics）

在 Python 中，相同的运算符作用于不同的数据类型时，其行为可能会有所不同。例如，字符串的加法表示拼接，而数字的加法表示求和。

str_result = "Hello, " + "world!" # 输出："Hello, world!"
num_result = 2 + 3                     # 输出：5

六、短路求值（Short-Circuit Evaluation）

逻辑运算符 `and` 和 `or` 在 Python 中执行短路求值。这意味着如果第一个表达式的结果已经能够确定整个表达式的值，就不会计算第二个表达式。

def yes(): return True
def no(): return False
def crash(): return 1/0 # 这会引发错误
print(no() and crash()) # 输出：False，不会执行 crash()
# print(yes() or crash()) # 输出：True，不会执行 crash()

七、type() vs isinstance()

`type()` 和 `isinstance()` 都可以用来检查对象的类型，但它们的工作方式不同。`type()` 检查对象的精确类型，而 `isinstance()` 检查对象是否是给定类型的实例，包括子类。

print(isinstance(2, int))    # 输出：True
print(isinstance(2.2, float)) # 输出：True
print(isinstance(True, bool)) # 输出：True
print(isinstance([1, 2], list)) # 输出：True

八、总结（Summary）

  • Python 有多种内置数据类型，如 int、float、bool 和 type。
  • 内置常数如 True、False、None 以及数学常数 math.pi 和 math.e。
  • Python 提供了丰富的运算符，包括算术、关系、赋值、逻辑运算符。
  • 运算符的优先级和结合律决定了表达式的计算顺序。
  • 类型会影响运算符的语义。
  • `and` 和 `or` 运算符执行短路求值。
  • `type()` 和 `isinstance()` 可以用来检查对象的类型，但用法有所不同。
