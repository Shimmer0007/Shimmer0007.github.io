Chapter 0 安装 Installation

本章节帮助学习者搭建本地Python开发环境，并提供了云端环境的备选方案。

安装清单

    课程主要对三种搭建学习环境所需的软件和工具进行介绍：

  • Miniconda：可以轻松创建和管理多个环境，确保不同项目之间的依赖关系互不干扰。推荐使用默认配置安装。以Windows系统为例，推荐下载链接：Miniconda3-latest-Windows-x86_64.exe
  • Visual Studio Code：（非常强大的）代码编辑器，支持丰富的插件。
  • Git：版本控制系统，用于下载课程资料/参与开源项目。

课程环境搭建

鉴于苯人先前试图为VSC引入R内核已经把环境变量整得乱成了一锅粥，直接在VSC里把exe文件路径设置好，配合装上的扩展凑合着用不失为一种逃课手段(×)。

坏消息是，全局Python环境毕竟是不利于项目隔离的，因为不同项目可能需要不同的依赖和配置（如，某个库的特定版本），项目的可移植性便显著下降。好消息是，Conda与VSC等IDE无缝集成，可以在VSC中直接激活和使用。

创建与激活Conda环境

创建命令：conda create -n p2s python=3.10

激活命令：conda activate p2s

删除环境命令：conda remove -n p2s --all

在VSC中：进入设置，搜索“Python: Select Interpreter”，选择刚刚创建的Conda环境。
